# SLR Bot — notslash

Variante do [SLR_bot](https://github.com/001zk/SLR_bot) sem slash commands — implementação dos mesmos comandos usando prefixo de texto em vez da Interactions API do Discord.

---

## Stack

- **JavaScript** · Node.js · discord.js
- Comandos com prefixo (ex: `!comando`)

---

## Contexto

Desenvolvido para comparar as duas abordagens de comandos no Discord: slash commands vs. prefix commands. Serviu para entender as diferenças de implementação, permissões e experiência do usuário entre os dois modelos.
